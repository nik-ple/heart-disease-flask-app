This web applications predicts whether a person has heart disease or not based on medical data provided by the user

Project Purpose

This project is built to help individuals and medical professionals to identify the likelihood of heart disease in a patient. So by using medical data of patient the ML model can generate predictions and can be helpful in early diagnosis of Heart Disease. and also helping in planning treatment.

Dataset

The dataset used to train the model is taken from UCI Machine Learning Repository and has 14 features like age, chest pain type, heartrate, serum cholesterol and may more.

Model Details

The Prediction model that is used is a machine learning regression model built using python. Many algorithms were considered and evaluated like K-Neighbours algorithm, Randoom forest classifier and Logistic regression; In which the Logistic Regression model turned out to be best model of them all. Then Logistic Regression model was fine-tuned using RandomisedSearchCV.

Features and Technologies

  • Python: The programming language used for developing model and web application.
  • Flask: A micro web framework used to develop web application.
  • HTML and CSS:Markup and Styling language used to develop interface.
  • Scikit-Learn: ML library used to develop algorithm for prediction.

How the app works:

The app allows users to input their information through form. Then the information is passed through the server to the model and the model; where the model processes the data and the model generates prediction.
