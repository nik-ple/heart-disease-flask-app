About PulsePrism

Multi-Angle Assessment of Cardiac Wellness

🎯 Why I Built PulsePrism

PulsePrism started with a simple goal: make early heart disease detection more accessible. Not everyone has immediate access to healthcare, but most people have access to the internet. So I asked myself — what if I could offer a meaningful prediction based on basic clinical data, right from your browser?

I wanted a tool that didn’t scare people, but informed them. Something clean, data-driven, and trustworthy.

📊 The Dataset That Powers It

PulsePrism is built on the Cleveland Heart Disease dataset — a real, medically validated dataset used by researchers for decades. It includes 14 features like age, cholesterol, blood pressure, chest pain type, and more. I cleaned the data, removed inconsistencies, and scaled it for model input so it performs well without biasing on any single metric.

🧠 What the Model Does (and Why We Chose It)

I tried various algorithms — KNN, Random Forest, Decision Trees. But Logistic Regression stood out. Not only did it deliver high accuracy (87.1%), it was also interpretable. That mattered to me, because I didn’t want users to get “black box” results.

Using techniques like RandomizedSearchCV, I tweaked the parameters and got strong performance on recall and AUC. That means it doesn’t just predict well — it actually helps detect real risks.

🚀 Tech Stack

Everything runs inside a Flask app, wrapped in Docker for deployment. The ML model is exposed via REST API. I host it on Render, and every Git commit can trigger a redeploy automatically.

  • Python & Flask – For model backend
  • Scikit-learn – Core ML toolkit
  • Docker – Portability & consistency
  • Render – Hosting the live version

👥 Who It’s For

I built this for students, developers, clinicians, and even curious individuals. You don’t need to be a data scientist or a doctor — if you can read your health reports, you can use PulsePrism.

Just remember, it’s a guide, not a diagnosis. Always follow up with a real doctor. But if PulsePrism makes you ask one more important question about your heart, I’ve done my job.
